🧱 Kanban Board Backend

📖 Описание
RESTful бэкенд для Kanban-доски, разработанный на Spring Boot.
Поддерживает аутентификацию через JWT, управление досками, колонками и задачами, а также реальное время (WebSocket) и полный аудит изменений.

Проект идеально подходит для изучения современных подходов к построению enterprise-приложений на Java.

✨ Возможности
🔐 Аутентификация и авторизация
JWT-токены, Spring Security, ограничение доступа по ролям.

📋 Управление досками
Создание, редактирование, удаление досок, добавление участников.

📌 Колонки и задачи
Drag-and-drop перемещение с умным пересчётом позиций.

⚡ Real‑time обновления
WebSocket (STOMP) – все изменения моментально приходят на клиент.

🕒 Аудит действий
Spring Data Envers сохраняет историю изменений каждой сущности.

🗄️ Миграции БД
Liquibase / Flyway – версионирование схемы.

🧪 Тестирование
Юнит-тесты, интеграционные тесты с Testcontainers.

🛠 Технологии
Компонент	Технология

Фреймворк	Spring Boot (Web, Data JPA, Security, WebSocket)
Аутентификация	JWT 
Аудит	Spring Data Envers
